API Documentation

Base URLs

These are the base URLs for the API:

  • Tracks: https://mvmapi.olk1.com/tracks
  • Albums: https://mvmapi.olk1.com/albums
  • Artwork: https://mvmapi.olk1.com/artwork

API Endpoints

1. /tracks

Fetch track information from the API.

Query Parameters:
  • ?_limit=number (Optional) Limits the number of results returned.
Response Fields:
  • id (Integer): Unique identifier for the track.
  • trackName (String): The name of the track.
  • albumName (String): The name of the album the track belongs to.
  • artistName (String): The name of the artist.
  • trackNumber (Integer): The track number on the album.
  • releaseYear (Integer): The release year of the track.
  • trackDuration (String): The duration of the track (in a time format like mm:ss).
  • albumDuration (String): The total duration of the album.
  • artworkUrl (String): URL of the track's artwork image.
  • youtubeUrl (String): URL of the track's music video on YouTube.
  • youtubePlaylistUrl (String): URL of the playlist the track is part of on YouTube.
  • genre (String): Genre of the track.

Example Requests:

  • GET https://mvmapi.olk1.com/tracks
  • GET https://mvmapi.olk1.com/tracks/1 (Fetch specific track by ID)
  • GET https://mvmapi.olk1.com/tracks?_limit=50 (Limit results to 50)

2. /albums

Fetch album information from the API.

Query Parameters:
  • ?_limit=number (Optional) Limits the number of results returned.
Response Fields:
  • id (Integer): Unique identifier for the album.
  • artist (String): Name of the artist who created the album.
  • title (String): The title of the album.
  • year (Integer): The release year of the album.
  • artwork (String): URL of the album artwork.
  • youtubeUrl (String): URL to the albumâ€™s music videos or playlist on YouTube.

Example Requests:

  • GET https://mvmapi.olk1.com/albums
  • GET https://mvmapi.olk1.com/albums/5 (Fetch specific album by ID)
  • GET https://mvmapi.olk1.com/albums?_limit=10 (Limit results to 10)

3. /artwork

Fetch artwork information from the API.

Query Parameters:
  • ?_limit=number (Optional) Limits the number of results returned.
Response Fields:
  • id (Integer): Unique identifier for the artwork.
  • artwork (String): URL to the image of the artwork.
  • youtubeUrl (String): YouTube URL for the artwork, if available.

Example Requests:

  • GET https://mvmapi.olk1.com/artwork
  • GET https://mvmapi.olk1.com/artwork/10 (Fetch specific artwork by ID)
  • GET https://mvmapi.olk1.com/artwork?_limit=5 (Limit results to 5)
